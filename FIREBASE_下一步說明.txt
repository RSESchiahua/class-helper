班級助手 Firebase 正式版 v1
確認關鍵字：HUA_FIREBASE_CLOUD_SYNC_CORE_20260711

這一版已加入：
1. Google 登入
2. Firebase Realtime Database 雲端備份
3. 手機與桌機共用同一份 localStorage 班級資料
4. 聯絡簿、簿本、積分、學生名單即時更新
5. 斷線時先保存本機，恢復網路後再同步
6. 完整 JSON 備份下載

重要：目前資料庫仍是鎖定模式，所以第一次部署後會顯示「待授權」。
請先在「目前保存既有資料的桌機」登入 Google，按「複製我的 UID」，再把 UID 傳給 ChatGPT。
在 UID 規則完成、桌機顯示「已同步」以前，先不要用手機建立或改資料。

本機測試：
1. 在專案終端機執行 npm install
2. 執行 npm run dev
3. 開啟終端機顯示的 localhost 網址

建立完成後：
- 先在桌機確認原有資料都還在
- 再登入 Google
- 複製 UID
- 等待資料庫規則完成
- 桌機第一次顯示「已同步」後，再用手機登入
